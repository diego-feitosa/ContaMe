# ContaMe
Projeto para auxiliar no controle de gastos

## Objetivo e as metas do projeto
Permitir que o(s) usuário(s) da aplicação possam ter controle de suas arrecadações e gastos. Cada valor de entrada e saída de recursos serão registrados no sistema e devem possuir informações relevantes para rastreabilidade, montagem de gráficos e geração de relatórios.

## Metodologia/estrutura de projeto escolhida
Projeto utilizará o básico da metodologia ágil. A aplicação funcionará em ambiente Web com tecnologias Open Sources e no padrão Rest API para que os dados possam ser consultados sem que toda a página seja atualizada.

## Entregas finais e critérios de aceitação
Projeto entregará ao usuário, um sistema que para auxiliar o controle financeiro, onde será sempre realizado cálculos dentro de certos critérios selecionados pelo cliente, como os custos mensais ou ganhos por investimentos, sejá por relatórios ou gráficos apropriados.
